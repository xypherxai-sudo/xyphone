XYPHER / Cal.com Integration

📅 Cal.com Integration einrichten

Verbinde Xypher mit deinem Cal.com Konto damit Lisa echte Termine buchen kann.

1
Cal.com Konto erstellen
Kostenlos starten — kein Kreditkarte nötig

Gehe zu cal.com und erstelle ein kostenloses Konto. Cal.com ist ein Open-Source Terminbuchungssystem.

💡 Tipp: Du kannst Cal.com auch selbst hosten. Für den Anfang reicht das kostenlose Cloud-Angebot.
2
Event Type erstellen
Definiere welche Dienstleistungen buchbar sind

Im Cal.com Dashboard:

  1. Klicke auf "+ New" → "Event Type"
  2. Gib einen Namen ein (z.B. "Haarschnitt" oder "Allgemeine Beratung")
  3. Stelle die Dauer ein (z.B. 30 oder 60 Minuten)
  4. Konfiguriere deine Verfügbarkeit
  5. Speichere und notiere die Event Type ID (steht in der URL: /event-types/123)
3
API Key generieren
Für den Datenzugriff

Im Cal.com Dashboard:

  1. Gehe zu Settings → Developer → API Keys
  2. Klicke auf "+ Add"
  3. Gib dem Key einen Namen (z.B. "Xypher")
  4. Kopiere den generierten API Key
4
Xypher konfigurieren
API Key und Event Type ID eintragen

Trage deine Cal.com Daten hier ein oder bearbeite die Datei calendar-config.json direkt:

✅ Konfiguration gespeichert! Starte den Server neu.

Oder direkt die Datei bearbeiten:

// /Users/admin/.openclaw/workspace/xypher/calendar-config.json
{
  "mode": "live",
  "calcom_api_key": "cal_live_DEIN_KEY",
  "event_type_id": "123",
  "business_name": "Mein Salon",
  "timezone": "Europe/Berlin"
}
5
Server neu starten & testen
Integration überprüfen
pkill -f "node server.js"
cd /Users/admin/.openclaw/workspace/xypher
node server.js &

# Test: Verfügbare Slots
curl "http://localhost:3131/api/calendar/slots?date=2026-04-01"

# Test: Konfiguration
curl "http://localhost:3131/api/calendar/config"

Wenn alles funktioniert, siehst du "mode": "live" in der Config-Antwort.
